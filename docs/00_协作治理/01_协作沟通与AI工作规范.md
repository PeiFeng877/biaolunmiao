# 协作沟通与AI工作规范

[PROTOCOL]: 变更时更新此头部，然后检查 agents.md

**版本**: v1.0
**日期**: 2026-02-17

## 1. 语言与输出规范
1. 与用户沟通统一使用中文。
2. 项目文档统一使用中文。
3. 思考与代码实现统一使用英文。
4. 新增文档文件名统一使用中文（`agents.md` 除外）。

## 2. AI 协作模式
1. 集成任务使用“跨端模式”：同一任务内协调 iOS + 后端 + 契约文档。
2. 专项任务可在子目录独立推进，但必须回写统一契约文档。
3. 跨端冲突以 `docs/03_接口与数据契约/` 为最终判定依据。

## 3. 执行顺序
1. 先确认范围与影响路径。
2. 先更新契约再实现代码。
3. 实现后执行最小可用测试。
4. 回写文档并标注版本、日期、变更日志。

## 4. 禁止事项
1. 不允许只改代码不改文档。
2. 不允许在未定义接口契约时直接跨端联调。
3. 不允许在生产环境直接手改配置绕过流程。
