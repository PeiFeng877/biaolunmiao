# Button Component Spec

[PROTOCOL]: 变更时更新此头部，然后检查 agents.md

**版本**: v1.0
**日期**: 2026-02-08
**范围**: `BianLunMiao/Views` 所有可点击按钮交互。

## 1. 组件白名单
- `AppButton(.primary)`
- `AppButton(.secondary)`
- `AppButton(.compactSecondary)`
- `AppButton(.ghost)`
- `AppButton(.toolbarText)`
- `AppIconButton(.topBarIcon)`
- `AppRowTapButton`
- `AppMenuAction`
- `AppTopBarButton(.topBarIcon)`
- `AppDetailTopBar`

## 2. 禁止项
- 业务层禁止直接使用裸 `Button(`。
- 业务层禁止 `.buttonStyle(.plain)`。
- 业务层禁止定义新的按钮样式类型，必须复用设计系统入口。

## 3. 状态规范
- 默认态: 使用对应 variant 的基础视觉。
- 按下态: 统一走 DesignSystem 的物理位移与触感反馈。
- 禁用态: 使用 `.disabled(...)` + 语义可见降权（如 opacity）。
- 危险态: 文案语义走 `role: .destructive`，样式走既有 variant。

## 4. 映射规则
- 主行动作: `AppButton(.primary)`。
- 次行动作: `AppButton(.secondary)`。
- 紧凑次行动作: `AppButton(.compactSecondary)`。
- 取消/弱动作: `AppButton(.ghost)`。
- 顶栏文本动作: `AppButton(.toolbarText)`。
- 顶栏图标动作: `AppIconButton` 或 `AppTopBarButton`。
- 详情页顶部导航: `AppDetailTopBar`（左右操作按钮必须同构）。
- 行级点击容器: `AppRowTapButton`。
- 菜单/确认动作: `AppMenuAction`。
