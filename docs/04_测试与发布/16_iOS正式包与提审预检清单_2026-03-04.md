## 4. 建议执行顺序
1. 发布技术支持页并回填 App Store Connect。
2. 在 App Store Connect 绑定本轮 `1.0.0 (4)` 构建。
3. 复核截图、隐私政策、审核备注与中国大陆供应信息（如本轮开启）。
4. 如果本轮要开中国大陆供应，确认真机中备案展示位可见后再提交。

## 5. 发包前强制检查
1. 运行治理脚本：`swift docs/03_Governance/tools/governance_audit.swift --mode check`
2. 运行 iOS 测试：至少覆盖 `BianLunMiaoTests`，并补一次真机 Apple 登录人工验收。
3. 确认 Release 包没有测试环境变量、debug token、临时域名与调试开关。
4. 在 App Store Connect 复核：
   - 隐私政策 URL
   - 技术支持 URL
   - 年龄分级
   - 审核备注
   - 中国大陆供应信息（如本轮开启）

1. `swift docs/03_Governance/tools/governance_audit.swift --mode generate`
   - 结果：已刷新 `按钮使用清单.md` 与 `反馈使用清单.md`

1. 如果仅提交全球版本且暂不开中国大陆供应，当前主要缺口是技术支持 URL 回填；本轮正式归档与导出物已生成。
